Hypertext Markup Language (HTML) is the 
standard markup language for documents designed 
to be displayed in a web browser.It is typically assisted by 
CSS for styling and JavaScript for animations.

Web browser takes HTML documents from a web 
server or from local storage and render the documents 
into web pages.
